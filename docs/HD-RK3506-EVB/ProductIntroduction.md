# HD-RK3506-EVB产品介绍

  **HD-RK3506-EVB** 是万象奥科针对 **HD-RK3506G-CORE** 系列核心板自主设计开发的标准评估底板，可用于核心板功能验证、应用开发以及产品方案评估。

<img src={require('./images/productintroduction-01.png').default} alt="47064f35-a18a-4ba0-b420-909fa6039b8b.png" style={{display: 'block', margin: '20px auto', maxWidth: '100%', borderRadius: '8px', boxShadow: '0 4px 8px rgba(0,0,0,0.1)'}}/>

  评估底板集成了丰富的外设接口，包括双路百兆以太网、USB、TF 卡、RGB LCD 显示接口、MIPI LCD 接口、GPIO、UART、调试接口等，可满足工业控制、人工智能、人机交互、无人零售、边缘计算等多种应用场景的开发需求。

  用户可直接搭配 **HD-RK3506G-CORE** 核心板进行软件开发、驱动调试和系统验证。也可参考评估板硬件设计完成产品底板开发，从而有效降低硬件设计难度，缩短产品开发周期，提高研发效率。
