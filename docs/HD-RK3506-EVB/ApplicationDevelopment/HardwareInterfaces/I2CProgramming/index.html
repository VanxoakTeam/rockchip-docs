1.1 I2CзЎ¬д»¶жЎҶжһ¶вҖӢ

еӣҫ 10.1 I2CжҖ»зәҝжӢ“жү‘еӣҫ

  • еңЁдёҖдёӘиҠҜзүҮ(SoC)еҶ…йғЁпјҢжңүдёҖдёӘжҲ–еӨҡдёӘI2CжҺ§еҲ¶еҷЁ
  • еңЁдёҖдёӘI2CжҺ§еҲ¶еҷЁдёҠпјҢеҸҜд»ҘиҝһжҺҘдёҖдёӘжҲ–еӨҡдёӘI2Cи®ҫеӨҮ
  • I2CжҖ»зәҝеҸӘйңҖиҰҒ2жқЎзәҝпјҡж—¶й’ҹзәҝSCLгҖҒж•°жҚ®зәҝSDA
  • еңЁI2CжҖ»зәҝзҡ„SCLгҖҒSDAзәҝдёҠпјҢйғҪжңүдёҠжӢүз”өйҳ»

1.2 I2CиҪҜд»¶жЎҶжһ¶вҖӢ

еӣҫ 10.2 I2CиҪҜд»¶жЎҶжһ¶

  • е®ғжҳҜеӨ§зҲ·пјҢдёҚе…іеҝғеә•еұӮе®һзҺ°зҡ„з»ҶиҠӮ

  • е®ғеҸӘйңҖиҰҒи°ғз”Ёи®ҫеӨҮй©ұеҠЁзЁӢеәҸжҸҗдҫӣзҡ„жҺҘеҸЈ

  • AT24C02й©ұеҠЁпјҡ

  • е®ғзҹҘйҒ“AT24C02иҰҒжұӮзҡ„ең°еқҖгҖҒж•°жҚ®ж јејҸ

  • е®ғзҹҘйҒ“еҸ‘еҮәд»Җд№ҲдҝЎеҸ·жүҚиғҪи®©AT24C02жү§иЎҢж“ҰйҷӨгҖҒзғ§еҶҷе·ҘдҪң

  • е®ғзҹҘйҒ“жҖҺд№ҲеҲӨж–ӯж•°жҚ®жҳҜеҗҰзғ§еҶҷжҲҗеҠҹ

  • е®ғжһ„йҖ еҘҪдёҖзі»еҲ—зҡ„ж•°жҚ®пјҢеҸ‘з»ҷI2CжҺ§еҲ¶еҷЁ

  • I2CжҺ§еҲ¶еҷЁй©ұеҠЁ

  • е®ғж №жҚ®I2CеҚҸи®®еҸ‘еҮәеҗ„зұ»дҝЎеҸ·пјҡI2Cи®ҫеӨҮең°еқҖгҖҒI2CеӯҳеӮЁең°еқҖгҖҒж•°жҚ®

  • е®ғж №жҚ®I2CеҚҸи®®еҲӨж–ӯ

2. I2CеҚҸи®®вҖӢ

вҖғвҖғеҸӮиҖғиө„ж–ҷпјҡ

2.1 зЎ¬д»¶иҝһжҺҘвҖӢ

вҖғвҖғI2CеңЁзЎ¬д»¶дёҠзҡ„жҺҘжі•еҰӮдёӢжүҖзӨәпјҢдё»жҺ§иҠҜзүҮеј•еҮәдёӨжқЎзәҝSCL,SDAзәҝпјҢеңЁдёҖжқЎI2CжҖ»зәҝдёҠеҸҜд»ҘжҺҘеҫҲеӨҡI2Cи®ҫеӨҮпјҢжҲ‘д»¬иҝҳдјҡж”ҫдёҖдёӘдёҠжӢүз”өйҳ»пјҲж”ҫдёҖдёӘдёҠжӢүз”өйҳ»зҡ„еҺҹеӣ д»ҘеҗҺжҲ‘д»¬еҶҚиҜҙпјүгҖӮ

2.2 дј иҫ“ж•°жҚ®зұ»жҜ”вҖӢ

вҖғвҖғжҖҺд№ҲйҖҡиҝҮI2Cдј иҫ“ж•°жҚ®пјҢжҲ‘д»¬йңҖиҰҒжҠҠж•°жҚ®д»Һдё»и®ҫеӨҮеҸ‘йҖҒеҲ°д»Һи®ҫеӨҮдёҠеҺ»пјҢд№ҹйңҖиҰҒжҠҠж•°жҚ®д»Һд»Һи®ҫеӨҮдј йҖҒеҲ°дё»и®ҫеӨҮдёҠеҺ»пјҢж•°жҚ®ж¶үеҸҠеҲ°еҸҢеҗ‘дј иҫ“гҖӮ

дёҫдёӘдҫӢеӯҗпјҡ

дҪ“иӮІиҖҒеёҲпјҡеҸҜд»ҘжҠҠзҗғеҸ‘з»ҷеӯҰз”ҹпјҢд№ҹеҸҜд»ҘжҠҠзҗғд»ҺеӯҰз”ҹдёӯжҺҘиҝҮжқҘгҖӮ

  • еҸ‘зҗғпјҡ

  • иҖҒеёҲпјҡејҖе§ӢдәҶ(start)

  • иҖҒеёҲпјҡAпјҒжҲ‘иҰҒеҸ‘зҗғз»ҷдҪ пјҒ(ең°еқҖ/ж–№еҗ‘)

  • еӯҰз”ҹAпјҡеҲ°пјҒ(еӣһеә”)

  • иҖҒеёҲжҠҠзҗғеҸ‘еҮәеҺ»пјҲдј иҫ“пјү

  • Aж”¶еҲ°зҗғд№ӢеҗҺпјҢеә”иҜҘе‘ҠиҜүиҖҒеёҲдёҖеЈ°пјҲеӣһеә”пјү

  • иҖҒеёҲпјҡз»“жқҹпјҲеҒңжӯўпјү

  • жҺҘзҗғпјҡ

  • иҖҒеёҲпјҡејҖе§ӢдәҶ(start)

  • иҖҒеёҲпјҡBпјҒжҠҠзҗғеҸ‘з»ҷжҲ‘пјҒ(ең°еқҖ/ж–№еҗ‘)

  • еӯҰз”ҹBпјҡеҲ°пјҒ

  • BжҠҠзҗғеҸ‘з»ҷиҖҒеёҲпјҲдј иҫ“пјү

  • иҖҒеёҲж”¶еҲ°зҗғд№ӢеҗҺпјҢз»ҷBиҜҙдёҖеЈ°пјҢиЎЁзӨәж”¶еҲ°зҗғдәҶпјҲеӣһеә”пјү

  • иҖҒеёҲпјҡз»“жқҹпјҲеҒңжӯўпјү

жҲ‘д»¬е°ұдҪҝз”ЁиҝҷдёӘз®ҖеҚ•зҡ„дҫӢеӯҗпјҢжқҘи§ЈйҮҠдёҖдёӢIICзҡ„дј иҫ“еҚҸи®®пјҡ

  • иҖҒеёҲиҜҙејҖе§ӢдәҶпјҢиЎЁзӨәејҖе§ӢдҝЎеҸ·(start)
  • иҖҒеёҲжҸҗйҶ’жҹҗдёӘеӯҰз”ҹиҰҒеҸ‘зҗғпјҢиЎЁзӨәеҸ‘йҖҒең°еқҖе’Ңж–№еҗ‘(address/read/write)
  • иҖҒеёҲеҸ‘зҗғ/жҺҘзҗғпјҢиЎЁзӨәж•°жҚ®зҡ„дј иҫ“
  • ж”¶еҲ°зҗғиҰҒеӣһеә”пјҡеӣһеә”дҝЎеҸ·(ACK)
  • иҖҒеёҲиҜҙз»“жқҹпјҢиЎЁзӨәIICдј иҫ“з»“жқҹ(P)

2.3 IICдј иҫ“ж•°жҚ®зҡ„ж јејҸвҖӢ

1пјүеҶҷж“ҚдҪңвҖӢ

жөҒзЁӢеҰӮдёӢпјҡ

  • дё»иҠҜзүҮиҰҒеҸ‘еҮәдёҖдёӘstartдҝЎеҸ·
  • з„¶еҗҺеҸ‘еҮәдёҖдёӘи®ҫеӨҮең°еқҖ(з”ЁжқҘзЎ®е®ҡжҳҜеҫҖе“ӘдёҖдёӘиҠҜзүҮеҶҷж•°жҚ®)пјҢж–№еҗ‘(иҜ»/еҶҷпјҢ0иЎЁзӨәеҶҷпјҢ1иЎЁзӨәиҜ»)
  • д»Һи®ҫеӨҮеӣһеә”(з”ЁжқҘзЎ®е®ҡиҝҷдёӘи®ҫеӨҮжҳҜеҗҰеӯҳеңЁ)пјҢз„¶еҗҺе°ұеҸҜд»Ҙдј иҫ“ж•°жҚ®
  • дё»и®ҫеӨҮеҸ‘йҖҒдёҖдёӘеӯ—иҠӮж•°жҚ®з»ҷд»Һи®ҫеӨҮпјҢе№¶зӯүеҫ…еӣһеә”
  • жҜҸдј иҫ“дёҖеӯ—иҠӮж•°жҚ®пјҢжҺҘж”¶ж–№иҰҒжңүдёҖдёӘеӣһеә”дҝЎеҸ·пјҲзЎ®е®ҡж•°жҚ®жҳҜеҗҰжҺҘеҸ—е®ҢжҲҗ)пјҢз„¶еҗҺеҶҚдј иҫ“дёӢдёҖдёӘж•°жҚ®гҖӮ
  • ж•°жҚ®еҸ‘йҖҒе®Ңд№ӢеҗҺпјҢдё»иҠҜзүҮе°ұдјҡеҸ‘йҖҒдёҖдёӘеҒңжӯўдҝЎеҸ·гҖӮ
  • дёӢеӣҫпјҡзҷҪиүІиғҢжҷҜиЎЁзӨә"дё»вҶ’д»Һ"пјҢзҒ°иүІиғҢжҷҜиЎЁзӨә"д»ҺвҶ’дё»"

2пјүиҜ»ж“ҚдҪңвҖӢ

жөҒзЁӢеҰӮдёӢпјҡ

  • дё»иҠҜзүҮиҰҒеҸ‘еҮәдёҖдёӘstartдҝЎеҸ·
  • з„¶еҗҺеҸ‘еҮәдёҖдёӘи®ҫеӨҮең°еқҖ(з”ЁжқҘзЎ®е®ҡжҳҜеҫҖе“ӘдёҖдёӘиҠҜзүҮеҶҷж•°жҚ®)пјҢж–№еҗ‘(иҜ»/еҶҷпјҢ0иЎЁзӨәеҶҷпјҢ1иЎЁзӨәиҜ»)
  • д»Һи®ҫеӨҮеӣһеә”(з”ЁжқҘзЎ®е®ҡиҝҷдёӘи®ҫеӨҮжҳҜеҗҰеӯҳеңЁ)пјҢз„¶еҗҺе°ұеҸҜд»Ҙдј иҫ“ж•°жҚ®
  • д»Һи®ҫеӨҮеҸ‘йҖҒдёҖдёӘеӯ—иҠӮж•°жҚ®з»ҷдё»и®ҫеӨҮпјҢе№¶зӯүеҫ…еӣһеә”
  • жҜҸдј иҫ“дёҖеӯ—иҠӮж•°жҚ®пјҢжҺҘж”¶ж–№иҰҒжңүдёҖдёӘеӣһеә”дҝЎеҸ·пјҲзЎ®е®ҡж•°жҚ®жҳҜеҗҰжҺҘеҸ—е®ҢжҲҗ)пјҢз„¶еҗҺеҶҚдј иҫ“дёӢдёҖдёӘж•°жҚ®гҖӮ
  • ж•°жҚ®еҸ‘йҖҒе®Ңд№ӢеҗҺпјҢдё»иҠҜзүҮе°ұдјҡеҸ‘йҖҒдёҖдёӘеҒңжӯўдҝЎеҸ·гҖӮ

дёӢеӣҫпјҡзҷҪиүІиғҢжҷҜиЎЁзӨә"дё»вҶ’д»Һ"пјҢзҒ°иүІиғҢжҷҜиЎЁзӨә"д»ҺвҶ’дё»"

3пјүI2CдҝЎеҸ·вҖӢ

I2CеҚҸи®®дёӯж•°жҚ®дј иҫ“зҡ„еҚ•дҪҚжҳҜеӯ—иҠӮпјҢд№ҹе°ұжҳҜ8дҪҚгҖӮдҪҶжҳҜиҰҒз”ЁеҲ°9дёӘж—¶й’ҹпјҡеүҚйқў8дёӘж—¶й’ҹз”ЁжқҘдј иҫ“8ж•°жҚ®пјҢз¬¬9дёӘж—¶й’ҹз”ЁжқҘдј иҫ“еӣһеә”дҝЎеҸ·гҖӮдј иҫ“ж—¶пјҢе…Ҳдј иҫ“жңҖй«ҳдҪҚ(MSB)гҖӮ

  • ејҖе§ӢдҝЎеҸ·пјҲSпјүпјҡSCLдёәй«ҳз”өе№іж—¶пјҢSDAеұұй«ҳз”өе№іеҗ‘дҪҺз”өе№іи·іеҸҳпјҢејҖе§Ӣдј йҖҒж•°жҚ®гҖӮ
  • з»“жқҹдҝЎеҸ·пјҲPпјүпјҡSCLдёәй«ҳз”өе№іж—¶пјҢSDAз”ұдҪҺз”өе№іеҗ‘й«ҳз”өе№іи·іеҸҳпјҢз»“жқҹдј йҖҒж•°жҚ®гҖӮ
  • е“Қеә”дҝЎеҸ·(ACK)пјҡжҺҘж”¶еҷЁеңЁжҺҘж”¶еҲ°8дҪҚж•°жҚ®еҗҺпјҢеңЁз¬¬9дёӘж—¶й’ҹе‘ЁжңҹпјҢжӢүдҪҺSDA
  • SDAдёҠдј иҫ“зҡ„ж•°жҚ®еҝ…йЎ»еңЁSCLдёәй«ҳз”өе№іжңҹй—ҙдҝқжҢҒзЁіе®ҡпјҢSDAдёҠзҡ„ж•°жҚ®еҸӘиғҪеңЁSCLдёәдҪҺз”өе№іжңҹй—ҙеҸҳеҢ–

I2CеҚҸи®®дҝЎеҸ·еҰӮдёӢпјҡ

4пјүеҚҸи®®з»ҶиҠӮвҖӢ

  • еҰӮдҪ•еңЁSDAдёҠе®һзҺ°еҸҢеҗ‘дј иҫ“пјҹ

  • дё»иҠҜзүҮйҖҡиҝҮдёҖж №SDAзәҝж—ўеҸҜд»ҘжҠҠж•°жҚ®еҸ‘з»ҷд»Һи®ҫеӨҮпјҢд№ҹеҸҜд»Ҙд»ҺSDAдёҠиҜ»еҸ–ж•°жҚ®пјҢиҝһжҺҘSDAзәҝзҡ„еј•и„ҡйҮҢйқўеҝ…з„¶жңүдёӨдёӘеј•и„ҡпјҲеҸ‘йҖҒеј•и„ҡ/жҺҘеҸ—еј•и„ҡпјүгҖӮ

  • дё»гҖҒд»Һи®ҫеӨҮйғҪеҸҜд»ҘйҖҡиҝҮSDAеҸ‘йҖҒж•°жҚ®пјҢиӮҜе®ҡдёҚиғҪеҗҢж—¶еҸ‘йҖҒж•°жҚ®пјҢжҖҺд№Ҳй”ҷејҖж—¶й—ҙпјҹеңЁ9дёӘж—¶й’ҹйҮҢпјҡ

  • еүҚ8дёӘж—¶й’ҹз”ұдё»и®ҫеӨҮеҸ‘йҖҒж•°жҚ®зҡ„иҜқпјҢз¬¬9дёӘж—¶й’ҹе°ұз”ұд»Һи®ҫеӨҮеҸ‘йҖҒж•°жҚ®пјӣ

  • еүҚ8дёӘж—¶й’ҹз”ұд»Һи®ҫеӨҮеҸ‘йҖҒж•°жҚ®зҡ„иҜқпјҢз¬¬9дёӘж—¶й’ҹе°ұз”ұдё»и®ҫеӨҮеҸ‘йҖҒж•°жҚ®гҖӮ

  • еҸҢж–№и®ҫеӨҮдёӯпјҢжҹҗдёӘи®ҫеӨҮеҸ‘йҖҒж•°жҚ®ж—¶пјҢеҸҰдёҖж–№жҖҺж ·жүҚиғҪдёҚеҪұе“ҚSDAдёҠзҡ„ж•°жҚ®пјҹ

  • и®ҫеӨҮзҡ„SDAдёӯжңүдёҖдёӘдёүжһҒз®ЎпјҢдҪҝз”ЁејҖжһҒ/ејҖжјҸз”өи·Ҝ(дёүжһҒз®ЎжҳҜејҖжһҒпјҢCMOSз®ЎжҳҜејҖжјҸпјҢдҪңз”ЁдёҖж ·)пјҢеҰӮдёӢеӣҫпјҡ

3.1 SMBusжҳҜI2CеҚҸи®®зҡ„дёҖдёӘеӯҗйӣҶвҖӢ

вҖғвҖғSMBus: System Management BusпјҢзі»з»ҹз®ЎзҗҶжҖ»зәҝгҖӮ SMBusжңҖеҲқзҡ„зӣ®зҡ„жҳҜдёәжҷәиғҪз”өжұ гҖҒе……з”өз”өжұ гҖҒе…¶д»–еҫ®жҺ§еҲ¶еҷЁд№Ӣй—ҙзҡ„йҖҡдҝЎй“ҫи·ҜиҖҢе®ҡд№үзҡ„гҖӮ SMBusд№ҹиў«з”ЁжқҘиҝһжҺҘеҗ„з§Қи®ҫеӨҮпјҢеҢ…жӢ¬з”өжәҗзӣёе…іи®ҫеӨҮпјҢзі»з»ҹдј ж„ҹеҷЁпјҢEEPROMйҖҡи®Ҝи®ҫеӨҮзӯүзӯүгҖӮ SMBus дёәзі»з»ҹе’Ңз”өжәҗз®ЎзҗҶиҝҷж ·зҡ„д»»еҠЎжҸҗдҫӣдәҶдёҖжқЎжҺ§еҲ¶жҖ»зәҝпјҢдҪҝз”Ё SMBus зҡ„зі»з»ҹпјҢи®ҫеӨҮд№Ӣй—ҙеҸ‘йҖҒе’ҢжҺҘж”¶ж¶ҲжҒҜйғҪжҳҜйҖҡиҝҮ SMBusпјҢиҖҢдёҚжҳҜдҪҝз”ЁеҚ•зӢ¬зҡ„жҺ§еҲ¶зәҝпјҢиҝҷж ·еҸҜд»ҘиҠӮзңҒи®ҫеӨҮзҡ„з®Ўи„ҡж•°гҖӮ SMBusжҳҜеҹәдәҺI2CеҚҸи®®зҡ„пјҢSMBusиҰҒжұӮжӣҙдёҘж јпјҢSMBusжҳҜI2CеҚҸи®®зҡ„еӯҗйӣҶгҖӮ

SMBusжңүе“ӘдәӣжӣҙдёҘж јзҡ„иҰҒжұӮпјҹи·ҹдёҖиҲ¬зҡ„I2CеҚҸи®®жңүе“Әдәӣе·®еҲ«пјҹ

  • VDDзҡ„жһҒйҷҗеҖјдёҚдёҖж ·

  • I2CеҚҸи®®пјҡиҢғеӣҙеҫҲе№ҝпјҢз”ҡиҮіи®Ёи®әдәҶй«ҳиҫҫ12Vзҡ„жғ…еҶө

  • SMBusпјҡ1.8V~5V

  • жңҖе°Ҹж—¶й’ҹйў‘зҺҮгҖҒжңҖеӨ§зҡ„Clock Stretching

  • Clock Stretchingеҗ«д№үпјҡжҹҗдёӘи®ҫеӨҮйңҖиҰҒжӣҙеӨҡж—¶й—ҙиҝӣиЎҢеҶ…йғЁзҡ„еӨ„зҗҶж—¶пјҢе®ғеҸҜд»ҘжҠҠSCLжӢүдҪҺеҚ дҪҸI2CжҖ»зәҝ

  • I2CеҚҸи®®пјҡж—¶й’ҹйў‘зҺҮжңҖе°ҸеҖјж— йҷҗеҲ¶пјҢClock Stretchingж—¶й•ҝд№ҹжІЎжңүйҷҗеҲ¶

  • SMBusпјҡж—¶й’ҹйў‘зҺҮжңҖе°ҸеҖјжҳҜ10KHzпјҢClock Stretchingзҡ„жңҖеӨ§ж—¶й—ҙеҖјд№ҹжңүйҷҗеҲ¶

  • ең°еқҖеӣһеә”(Address Acknowledge)пјҡдёҖдёӘI2Cи®ҫеӨҮжҺҘж”¶еҲ°е®ғзҡ„и®ҫеӨҮең°еқҖеҗҺпјҢжҳҜеҗҰеҝ…йЎ»еҸ‘еҮәеӣһеә”дҝЎеҸ·пјҹ

  • I2CеҚҸи®®пјҡжІЎжңүејәеҲ¶иҰҒжұӮеҝ…йЎ»еҸ‘еҮәеӣһеә”дҝЎеҸ·

  • SMBusпјҡејәеҲ¶иҰҒжұӮеҝ…йЎ»еҸ‘еҮәеӣһеә”дҝЎеҸ·пјҢиҝҷж ·еҜ№ж–№жүҚзҹҘйҒ“иҜҘи®ҫеӨҮзҡ„зҠ¶жҖҒпјҡbusyпјҢfailedпјҢжҲ–жҳҜиў«з§»йҷӨдәҶ

  • SMBusеҚҸи®®жҳҺзЎ®дәҶж•°жҚ®зҡ„дј иҫ“ж јејҸ

  • I2CеҚҸи®®пјҡе®ғеҸӘе®ҡд№үдәҶжҖҺд№Ҳдј иҫ“ж•°жҚ®пјҢдҪҶжҳҜе№¶жІЎжңүе®ҡд№үж•°жҚ®зҡ„ж јејҸпјҢиҝҷе®Ңе…Ёз”ұи®ҫеӨҮжқҘе®ҡд№ү

3.2 SMBusеҚҸи®®еҲҶжһҗвҖӢ

вҖғвҖғеҜ№дәҺI2CеҚҸи®®пјҢе®ғеҸӘе®ҡд№үдәҶжҖҺд№Ҳдј иҫ“ж•°жҚ®пјҢдҪҶжҳҜе№¶жІЎжңүе®ҡд№үж•°жҚ®зҡ„ж јејҸпјҢиҝҷе®Ңе…Ёз”ұи®ҫеӨҮжқҘе®ҡд№үгҖӮ еҜ№дәҺSMBusеҚҸи®®пјҢе®ғе®ҡд№үдәҶеҮ з§Қж•°жҚ®ж јејҸгҖӮ

1пјүsymbols(з¬ҰеҸ·)вҖӢ

S     (1 bit) : Start bit(ејҖе§ӢдҪҚ)
Sr    (1 bit) : йҮҚеӨҚзҡ„ејҖе§ӢдҪҚ

                get a 10 bit I2C address.
                (ең°еқҖдҪҚпјҢ7дҪҚең°еқҖ)
Command Code  (8 bits): Command byte, a data byte which often selects a register on
                the device.
                (е‘Ҫд»Өеӯ—иҠӮпјҢдёҖиҲ¬з”ЁжқҘйҖүжӢ©иҠҜзүҮеҶ…йғЁзҡ„еҜ„еӯҳеҷЁ)

[..]:           Data sent by I2C device, as opposed to data sent by the host
                adapter.
                (дёӯжӢ¬еҸ·иЎЁзӨәI2Cи®ҫеӨҮеҸ‘йҖҒзҡ„ж•°жҚ®пјҢжІЎжңүдёӯжӢ¬еҸ·иЎЁзӨәhost adapterеҸ‘йҖҒзҡ„ж•°жҚ®)

4.1 йҮҚиҰҒз»“жһ„дҪ“вҖӢ

вҖғвҖғдҪҝз”ЁдёҖеҸҘиҜқжҰӮжӢ¬I2Cдј иҫ“пјҡAPPйҖҡиҝҮI2C ControllerдёҺI2C Deviceдј иҫ“ж•°жҚ®гҖӮ

еңЁLinuxдёӯиҰҒжҖқзҙўдёӢйқўеҮ дёӘй—®йўҳгҖӮ

  • жҖҺд№ҲиЎЁзӨәI2C Controller
  • дёҖдёӘиҠҜзүҮйҮҢеҸҜиғҪжңүеӨҡдёӘI2C ControllerпјҢжҜ”еҰӮз¬¬0дёӘгҖҒз¬¬1дёӘгҖҒвҖҰвҖҰ
  • еҜ№дәҺдҪҝз”ЁиҖ…пјҢеҸӘиҰҒзЎ®е®ҡжҳҜз¬¬еҮ дёӘI2C ControllerеҚіеҸҜ
  • дҪҝз”Ёi2c_adapterиЎЁзӨәдёҖдёӘI2C BUSпјҢжҲ–з§°дёәI2C ControllerпјҢйҮҢйқўжңү2дёӘйҮҚиҰҒзҡ„жҲҗе‘ҳпјҡ

вҖғвҖғa) nrпјҡз¬¬еҮ дёӘI2C BUS(I2C Controller)

вҖғвҖғb) i2c_algorithmпјҢйҮҢйқўжңүиҜҘI2C BUSзҡ„дј иҫ“еҮҪж•°пјҢз”ЁжқҘж”¶еҸ‘I2Cж•°жҚ®

i2c_adapterеҺҹеһӢпјҡ

i2c_algorithmеҺҹеһӢпјҡ

1пјүжҖҺд№ҲиЎЁзӨәI2C DeviceвҖӢ

  • дёҖдёӘI2C DeviceпјҢдёҖе®ҡжңүи®ҫеӨҮең°еқҖ
  • е®ғиҝһжҺҘеңЁе“ӘдёӘI2C ControllerдёҠпјҢеҚіеҜ№еә”зҡ„i2c_adapterжҳҜд»Җд№Ҳ

вҖғвҖғдҪҝз”Ёi2c_clientжқҘиЎЁзӨәдёҖдёӘI2C Device

2пјүжҖҺд№ҲиЎЁзӨәиҰҒдј иҫ“зҡ„ж•°жҚ®вҖӢ

вҖғвҖғеңЁдёҠйқўзҡ„i2c_algorithmз»“жһ„дҪ“дёӯеҸҜд»ҘзңӢеҲ°иҰҒдј иҫ“зҡ„ж•°жҚ®иў«з§°дёәпјҡi2c_msg

i2c_msgеҺҹеһӢпјҡ

  • i2c_msgдёӯзҡ„flagsз”ЁжқҘиЎЁзӨәдј иҫ“ж–№еҗ‘пјҡbit 0зӯүдәҺI2C_M_RDиЎЁзӨәиҜ»пјҢbit 0зӯүдәҺ0иЎЁзӨәеҶҷ
  • дёҖдёӘi2c_msgиҰҒд№ҲжҳҜиҜ»пјҢиҰҒд№ҲжҳҜеҶҷ

вҖғвҖғдёҫдҫӢпјҡи®ҫеӨҮең°еқҖдёә0x50зҡ„EEPROMпјҢиҰҒиҜ»еҸ–е®ғйҮҢйқўеӯҳеӮЁең°еқҖдёә0x10зҡ„дёҖдёӘеӯ—иҠӮпјҢеә”иҜҘжһ„йҖ еҮ дёӘi2c_msgпјҹиҰҒжһ„йҖ 2дёӘi2c_msg

вҖғвҖғвҖғвҖғc) з¬¬дёҖдёӘi2c_msgиЎЁзӨәеҶҷж“ҚдҪңпјҢжҠҠиҰҒи®ҝй—®зҡ„еӯҳеӮЁең°еқҖ0x10еҸ‘з»ҷи®ҫеӨҮ

вҖғвҖғвҖғвҖғd) з¬¬дәҢдёӘi2c_msgиЎЁзӨәиҜ»ж“ҚдҪң

вҖғвҖғд»Јз ҒеҰӮдёӢпјҡ

# i2cset
вҖғвҖғUsage: i2cset [-f] [-y] [-m MASK] [-r] [-a] I2CBUS CHIP-ADDRESS DATA-ADDRESS [VALUE] ... [MODE]
  I2CBUS is an integer or an I2C bus name

# i2ctransfer
вҖғвҖғUsage: i2ctransfer [-f] [-y] [-v] [-V] [-a] I2CBUS DESC [DATA] [DESC [DATA]]...
  I2CBUS is an integer or an I2C bus name
  DESC describes the transfer in the form: {r|w}LENGTH[@address]
    1) read/write-flag 2) LENGTH (range 0-65535) 3) I2C address (use last one if omitted)
  DATA are LENGTH bytes for a write message. They can be shortened by a suffix:
    = (keep value constant until LENGTH)
    + (increase value by 1 until LENGTH)
    - (decrease value by 1 until LENGTH)
    p (use pseudo random generator until LENGTH with value as seed)

Example (bus 0, read 8 byte at offset 0x64 from EEPROM at 0x50):
  # i2ctransfer 0 w1@0x50 0x64 r8
Example (same EEPROM, at offset 0x42 write 0xff 0xfe ... 0xf0):
  # i2ctransfer 0 w17@0x50 0x42 0xff-
